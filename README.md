# We Are ApplyLoop Sub

A modern dashboard for job applications built with Next.js and Tailwind CSS.

## Features

- Modern UI with responsive design
- Centralized Data Layer for easy backend integration
- Dynamic application management
- Interactive dashboard and settings
- Real-time data visualization potential

## Technology Stack

- Next.js
- React
- Tailwind CSS
- React Icons

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open your browser and navigate to `http://localhost:3000/dashboard`
